# Research: Evidence Weaving

## Sources
- UNC Writing Center: "Quotations" (integration techniques)
- Ivy Tech / PVCC: "Quote Sandwich" method
- Idaho Pressbooks: "When to Quote, Paraphrase, or Summarize"
- Cal State Pressbooks: "Integrating Source Evidence"

## Key Principles

1. **The setup-evidence-significance pattern.** Academic version: introduce → quote → analyze. Journalism version: create a narrative need → fill it with evidence → state the implication. The key difference: in journalism, the setup creates TENSION that the evidence resolves.

2. **Quote when voice matters, paraphrase when clarity matters.** "Functionally fictional" is worth quoting because no journalist would invent that phrase. "The temperature was 30 degrees" is not worth quoting because the number speaks for itself. Decision framework: would the reader lose something if you paraphrased? If yes, quote. If no, paraphrase.

3. **Never let evidence interrupt narrative flow.** Evidence should feel like a natural part of the paragraph's argument. If the reader feels the story "stop for a citation," the integration has failed. Embed data mid-sentence when possible: "a rate that makes the IPCC's carbon budget 'functionally fictional'" rather than stopping to cite.

4. **The 2/3 rule.** The writer's voice should dominate. Roughly 2/3 of the text should be the journalist's own words; 1/3 should be sourced evidence (quotes, data, documents). If evidence dominates, the piece reads as a transcript, not journalism.

5. **Don't quote routine facts.** Quotes are for opinions, emotions, and vivid language. Numbers, dates, and technical specifications should be paraphrased and attributed: "Petrova's team measured a 340% increase, she reported in Nature Geoscience" rather than quoting the number.
